## Requirements

iOS device that supports BlueTooth 4.0 & an [RFduino](http://rfduino.com) board.

## Installation

RFduino-library is available through [CocoaPods](http://cocoapods.org), to install
it simply add the following line to your Podfile:

    pod "RFduino-library"
